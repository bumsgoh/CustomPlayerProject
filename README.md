# MP4 local / HLS Streaming player


# Project CustomPlayer

AVPlayer에 커스터마이징이 불가능한 부분을 개선하기 위해서 VideoToolbox를 사용하여 만든 플레이어

## Getting Started

#### Local 파일(.mp4<H.264/AAC>) 재생 
 - MPEG-4 Parser 
 - NAL Parser (H.264) 
 
#### Remote server 파일(.m3u8, .ts<H.264/AAC>) 재생 
 - M3U8 Parser 
 - TS Parser 

### Tech stack

* VideoToolbox / AudioToolbox
* AVFoundation
